# Javascript and Typescript Exercises

This Exercices are server with Parcel. This is thought as utility libray.

## Install

For install the project, open a Terminal and run:

```bash
$ yarn
```

or

```bash
$ npm i
```

## Test

```bash
$ yarn test
```

or

```bash
$ npm test
```

## Serve

```bash
$ yarn start:prod
```

or

```bash
$ npm run start:prod
```
